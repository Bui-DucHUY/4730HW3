# Task Management Dashboard (4730HW3)
Homework 3: Building a web application frontend with Vue.js

## Features
- Full task CRUD (Create, Read, Update, Delete)
- Computed Search and Status Filtering
- LocalStorage persistence using Watchers
- Bonus: 25-Minute Pomodoro Focus Timer per task

## How to Run the Project
1. Navigate into the `hw3` directory:
   cd hw3
2. Install dependencies:
   npm install
3. Run the development server:
   npm run dev
